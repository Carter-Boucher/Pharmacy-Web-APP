## Starting the Website
  
#### /frontend
  `cd` into `frontend`
  
  `npm install` to install all dependencies
  
  `npm start` to compile and run web app
  
#### /backend
  run the `main.py` file in `/backend`
  
#### (optional)
  download DB Browser SQLite and open `sql_app.db`
